# Exercise 2.7
## What values do these literals represent? What type does each have?
### (a) "Who goes with F\145rgus?\012"
- value : "Who goes with Fergus?\n"
- type : `array` of constant `char`s
### (b) 3.14e1L
- value : 31.4
- type : `long double`
### (c) 1024f
- value : 1024
- type : float
### (d) 3.14L
- value : 3.14
- type : long double
